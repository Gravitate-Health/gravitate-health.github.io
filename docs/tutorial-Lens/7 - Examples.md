# Lens Examples

The following lenses have been developed for the Gravitate-Health platform and are available as open-source implementations.

| Lens Name | Description | Repository |
|-----------|-------------|------------|
| Pregnancy Lens | Highlights pregnancy, childbearing age or breastfeeding information | [lens-logic-pregnancy](https://github.com/Gravitate-Health/lens-logic-pregnancy) |
| Condition Lens | Highlights information about specific condition | [lens-logic-conditions](https://github.com/Gravitate-Health/lens-logic-conditions) |
| RMM Lens | Integrates Risk Minimization Measures | [lens-logic-risk-minimization-material](https://github.com/Gravitate-Health/lens-logic-risk-minimization-material) |
| Contact Lens | Adds GP or similar contact information directly into ePI | [lens-logic-contact](https://github.com/Gravitate-Health/lens-logic-contact) |
| Diabetes Lens | Adds checklist for diabetes medication | [lens-logic-diabetes](https://github.com/Gravitate-Health/lens-logic-diabetes) |
| Questionnaire Lens | Adds a questionnaire link | [lens-logic-questionnaire](https://github.com/Gravitate-Health/lens-logic-questionnaire) |
| Medication-Lab-Interaction Lens | Highlights interaction with medication and lab result | [lens-logic-med-lab](https://github.com/Gravitate-Health/lens-logic-med-lab) |
| Drug-Drug-Interaction Lens | Highlights drug-drug interaction | [lens-logic-interaction](https://github.com/Gravitate-Health/lens-logic-interaction) |
| Doping Lens | Provides warning regarding doping properties for professional athletes | [lens-logic-doping](https://github.com/Gravitate-Health/lens-logic-doping) |
| Lab Test Lens | Highlights information about renal adjustment with lab result (eGFR) | [lens-logic-test-lab](https://github.com/Gravitate-Health/lens-logic-test-lab) |
| Summary Lens | Summarizes ePI with AI | [lens-service-summary](https://github.com/Gravitate-Health/lens-service-summary) |
| Allergy and Intolerance Lens | Highlights allergy or intolerance information (e.g., lactose) | [lens-logic-allergy-and-intollerance](https://github.com/Gravitate-Health/lens-logic-allergy-and-intollerance) |
| Plain Language Lens | Provides plain language translation for difficult text | [lens-service-plain-language](https://github.com/Gravitate-Health/lens-service-plain-language) |
| Community Lens | Provides information about online communities | [gh-community-lens](https://github.com/joofio/gh-community-lens) |
