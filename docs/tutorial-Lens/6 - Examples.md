# Lens Examples

The following lenses have been developed for the Gravitate-Health platform and are available as open-source implementations.

| Lens Name | Description | Repository |
|-----------|-------------|------------|
| Pregnancy Lens | Highlights pregnancy, childbearing age or breastfeeding information | [pregnancy-lens](https://github.com/Gravitate-Health/pregnancy-lens) |
| Condition Lens | Highlights information about specific condition | [conditions-lens](https://github.com/Gravitate-Health/conditions-lens) |
| RMM Lens | Integrates Risk Minimization Measures | [rmm-lens](https://github.com/Gravitate-Health/rmm-lens) |
| Contact Lens | Adds GP or similar contact information directly into ePI | [gh-contact-lens](https://github.com/Gravitate-Health/gh-contact-lens) |
| Diabetes Lens | Adds checklist for diabetes medication | [diabetes-lens](https://github.com/Gravitate-Health/diabetes-lens) |
| Questionnaire Lens | Adds a questionnaire link | [gh-questionnaire-lens](https://github.com/Gravitate-Health/gh-questionnaire-lens) |
| Medication-Lab-Interaction Lens | Highlights interaction with medication and lab result | [gh-med-lab-lens](https://github.com/Gravitate-Health/gh-med-lab-lens) |
| Drug-Drug-Interaction Lens | Highlights drug-drug interaction | [interaction-lens](https://github.com/Gravitate-Health/interaction-lens) |
| Doping Lens | Provides warning regarding doping properties for professional athletes | [gh-doping-lens](https://github.com/Gravitate-Health/gh-doping-lens) |
| Lab Test Lens | Highlights information about renal adjustment with lab result (eGFR) | [gh-test-lab-lens](https://github.com/Gravitate-Health/gh-test-lab-lens) |
| Summary Lens | Summarizes ePI with AI | [summary-lens-service](https://github.com/Gravitate-Health/summary-lens-service) |
| Allergy and Intolerance Lens | Highlights allergy or intolerance information (e.g., lactose) | [allergy-and-intollerance-lens](https://github.com/Gravitate-Health/allergy-and-intollerance-lens) |
| Plain Language Lens | Provides plain language translation for difficult text | [plain-language-lens](https://github.com/Gravitate-Health/plain-language-lens) |
| Community Lens | Provides information about online communities | [gh-community-lens](https://github.com/joofio/gh-community-lens) |
