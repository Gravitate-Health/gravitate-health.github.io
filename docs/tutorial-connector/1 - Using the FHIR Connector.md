# Using the FHIR Connector

## Capabilities

:::info FHIR Connector
The FHIR connector synchronizes data against a remote FHIR endpoint.
:::

### Key Features

- **Remote synchronization**: Connects to external FHIR servers
- **Standard compliance**: Follows HL7 FHIR specifications
- **Data transformation**: Handles FHIR resource formatting
