# Examples and Resources

Existing connectors within FOSPS that can serve as references include:

*  *[FHIR Connector](https://github.com/Gravitate-Health/fhir-connector)*: Syncs resources from the HL7 FHIR server sandbox.
*  *[Lenses Connector](https://github.com/Gravitate-Health/lenses-connector)*: a Push-type connector which fetches lenses from github source code (already in FHIR format) and pushes them to the server.
